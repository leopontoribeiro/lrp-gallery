🚨 PROBLEMAS & SOLUÇÕES (SOS)
================================================================================

PROBLEMA: Admin não consegue fazer login
SOLUÇÃO:
1. Verificar email correto em auth.users
2. Verificar se user tem owner_id em clients
3. Se não: UPDATE clients SET owner_id = 'USER_ID' WHERE id = 'CLIENT_ID'
4. Limpar cookies/cache
5. Testar em navegador privado

PROBLEMA: QR code não aparece
SOLUÇÃO:
1. SELECT qr_code_url FROM share_links WHERE token = 'TOKEN'
2. Se NULL: UPDATE share_links SET qr_code_url = generate_qr_code_url(token)
3. Refresh página

PROBLEMA: Upload de imagem falha
SOLUÇÃO:
1. Verificar tamanho: Max 5MB
2. Verificar tipo: JPG, PNG, WebP
3. Verificar espaço em Storage
4. Se full: DELETE antigos ou aumentar quota

PROBLEMA: Performance lenta
SOLUÇÃO:
1. Abrir Sentry > Performance tab
2. Identificar endpoint lento
3. Rodar: EXPLAIN ANALYZE SELECT...
4. Adicionar índice se necessário
5. Aumentar connection pool

PROBLEMA: Evento não foi deletado após 180 dias
SOLUÇÃO:
1. Verificar auto_delete_enabled = true
2. Verificar GitHub Actions logs
3. Se cron falhou, deletar manualmente:
   SELECT delete_event_and_galleries('EVENT_ID'::uuid);
4. Notificar legal team (LGPD violation)
