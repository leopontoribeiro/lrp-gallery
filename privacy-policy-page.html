2. Uso de Imagens Biométricas

2.1 O que é Dado Biométrico?

Dados biométricos são características físicas ou comportamentais únicas de uma pessoa, como reconhecimento facial, impressão digital ou íris. No nosso caso, utilizamos reconhecimento facial através de selfies.

2.2 Como Coletamos Suas Imagens

  1. Consentimento Prévio: Você recebe um modal informando exatamente como seus dados serão usados
  2. Captura Voluntária: Você tira uma selfie com seu próprio dispositivo
  3. Processamento Local (Opcional): A imagem pode ser processada no seu telefone antes de enviar
  4. Envio Seguro: Transmissão criptografada (TLS 1.3) para nossos servidores
  5. Armazenamento Seguro: Criptografia AES-256-GCM em repouso

2.3 Propósitos de Uso

✓ AUTORIZADO - O que fazemos com sua selfie:

  • Identificar suas fotos no álbum do evento
  • Melhorar a precisão do algoritmo de reconhecimento facial
  • Operações de segurança (prevenção de fraudes)
  • Cumprir obrigações legais e regulatórias

✗ PROIBIDO - O que NÃO fazemos:

  • Compartilhar com terceiros
  • Usar para marketing ou publicidade
  • Vender seus dados
  • Usar para vigilância ou rastreamento
  • Combinar com outras bases de dados
  • Usar para análise comportamental não autorizada

2.4 Tipos de Dados Coletados

Tipo de Dado Sensibilidade Armazenamento Retenção
Selfie (imagem) 🔴 CRÍTICA Criptografada 180 dias
Face embedding* 🟠 ALTA Criptografada 180 dias
Timestamp 🟢 BAIXA Log auditado 2 anos
IP/User-Agent 🟡 MÉDIA Log auditado 90 dias

*Face embedding = representação matemática do rosto (NÃO é a imagem original)

3. Conformidade LGPD ✅ CONFORME

3.1 Bases Legais para Processamento

O processamento de seus dados biométricos é realizado com base em:

Lei Geral de Proteção de Dados (Lei 13.709/2018)

  • Art. 7, II: Consentimento livre, informado, específico e inequívoco
  • Art. 8: Autorização especial para tratamento de dados biométricos
  • Art. 7, VI: Interesse legítimo para operações de segurança

3.2 Princípios LGPD Respeitados

1. FINALIDADE Dados usados APENAS para reconhecimento facial no evento. Nenhum outro uso.
2. ADEQUAÇÃO Mínimos dados necessários coletados. Apenas selfie, nada mais.
3. TRANSPARÊNCIA Você sabe o QUÊ, COMO, QUANDO, POR QUÊ e QUEM tem acesso.
4. SEGURANÇA Criptografia de ponta a ponta com padrões internacionais.
5. LIVRE ACESSO Você pode acessar, corrigir, deletar ou portar seus dados.
6. QUALIDADE Dados mantidos precisos e não obsoletos. Você controla.

3.3 Consentimento Prévio e Informado

  1. Antes de qualquer processamento, você recebe um modal claro em português
  2. Explica EXATAMENTE o que será feito com sua selfie
  3. Você marca 3 checkboxes obrigatórios confirmando que:
    • Entende como seus dados serão usados
    • Reconhece seus direitos LGPD
    • Consente explicitamente com o processamento
  4. Consentimento é registrado com timestamp, IP e user-agent
  5. Você pode revogar a QUALQUER MOMENTO

5. Segurança de Dados

5.1 Criptografia

Localização Método Padrão
Em Trânsito (Internet) TLS 1.3 Padrão IETF
Em Repouso (Servidor) AES-256-GCM NIST Aprovado
Gerenciamento de Chaves HSM (Hardware Security Module) FIPS 140-2 Nível 3

5.2 Controle de Acesso

Como protegemos seu acesso:

  • 🔐 Autenticação Multi-Fator (MFA): Múltiplas formas de verificação
  • 🔐 Sessões Limitadas: Acesso administrativo expira após 1 hora
  • 🔐 IP Whitelisting: Apenas IPs autorizados acessam dados
  • 🔐 Role-Based Access: Cada pessoa vê apenas o necessário

5.3 Infraestrutura Segura

Certificações e Conformidade:

  • ISO 27001 (Segurança da Informação)
  • SOC 2 Type II (Relatório de Conformidade)
  • HIPAA Ready (Saúde/Privacidade)
  • Data centers com redundância geográfica

5.4 Monitoramento e Detecção

  • ✓ Monitoramento 24/7 de acessos anormais
  • ✓ Alertas de segurança em tempo real
  • ✓ Testes de penetração semestrais
  • ✓ Auditorias de segurança anuais
  • ✓ Plano de resposta a incidentes documentado

5.5 O Que Você Pode Fazer

  • Use senhas fortes e únicas para sua conta
  • Não compartilhe seu link de acesso
  • Faça logout depois de usar
  • Mantenha seu dispositivo protegido

8. Perguntas Frequentes (FAQ)

Minha selfie será salva no meu telefone? ▼
A selfie pode ser processada no seu telefone para melhor privacidade, mas é enviada aos nossos servidores seguros para reconhecimento facial. Você controla se mantém uma cópia local.
Por quanto tempo minha selfie fica armazenada? ▼
180 dias (6 meses) a partir do dia do evento. Você recebe notificação 30 dias antes de ser deletada. Pode solicitar exclusão imediata a qualquer momento.
Vocês compartilham minha selfie com terceiros? ▼
NÃO. Nunca compartilhamos dados biométricos com terceiros. A selfie é usada APENAS para sua busca de fotos no evento.
Como meus dados são protegidos? ▼
Usamos criptografia AES-256-GCM, autenticação multi-fator, data centers certificados ISO 27001, e monitoramento 24/7. Seus dados são algumas das informações mais protegidas que temos.
Posso deletar minha selfie antes de 180 dias? ▼
Sim! Você pode solicitar exclusão imediata a qualquer momento. Acesse o portal, clique em "Deletar Meu Rosto" ou envie email. Será deletada em até 30 dias.
O que é face embedding? ▼
É uma representação matemática do seu rosto (números, não a imagem). A partir do face embedding, é IMPOSSÍVEL reconstruir sua selfie. É como uma senha do seu rosto.
Vocês usam AI/Machine Learning com minha selfie? ▼
Sim, para melhorar a precisão do reconhecimento facial. VOCÊ PODE RECUSAR ISSO durante o consentimento ou exercendo seu Direito de Oposição.
E se eu não consentir? ▼
Sem consentimento, você não pode usar o recurso de busca por rosto. Mas pode: (1) visualizar fotos manualmente, (2) consentir depois, (3) usar o álbum normalmente.
Como exerço meu direito de revogação? ▼
1 clique no app ("Configurações" > "Revogar Consentimento") ou envie email. Efetivo IMEDIATAMENTE. Você pode consentir novamente depois se desejar.
Qual é seu Encarregado de Proteção de Dados (DPO)? ▼
Leandro Ribeiro (DPO - Data Protection Officer). Contato: souleandroribeiro@gmail.com. Ele é responsável por garantir conformidade LGPD em todos os processos.
O que fazer se meus dados foram vazados? ▼
Contato IMEDIATO: souleandroribeiro@gmail.com com "VAZAMENTO DE DADOS". Investigaremos, notificaremos você e a ANPD conforme lei. Você será atualizado diariamente.
